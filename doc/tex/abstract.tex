\abstract{
	Akinator é um gênio virtual que é capaz de adivinhar a personagem em que o usuário está pensando, seja ela real ou fictícia, através de perguntas sobre suas características (\url{http://www.akinator.com}). Neste exercício usaremos Prolog para modelar uma base de conhecimento, e implementar uma mini-versão do jogo Akinator. Além disso, serão comparados varias ordens das características e das regras das personagens em termos do número medio de perguntas.
}
